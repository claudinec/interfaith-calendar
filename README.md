# Interfaith calendar

Data and functions.

Fixed dates.

Moveable dates.

## Data specification

`religions.csv`: List of religions (machine name and human names).

`<religion>.yaml`: Dates for each religion.

```yaml
Primary name:
month:
day:
  alternative names:
  calendar: Gregorian (default) or Julian
  hemisphere: North or South
  description:
  notes:
```
